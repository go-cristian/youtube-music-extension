# Privacy Policy

YouTube Music Floating Player does not collect, store, sell, or transmit personal data.

The extension runs only on `https://music.youtube.com/` to provide playback controls and a themed mini player. It reads playback state from the active YouTube Music page only to display the current track and control playback.

The extension stores only the selected theme preference locally in the browser using Chrome storage. This setting is not transmitted to any server.

The extension does not use analytics, tracking, advertising, remote code, or third-party data services.
